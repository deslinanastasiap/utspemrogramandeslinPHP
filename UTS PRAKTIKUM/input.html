Masukan Panjang dan Lebar Persegi Panjang

Percabangan digunakan untuk memeriksa hasil input.Input yang diperbolehkan adalah yang bernilai lebih dari 0 (tidak bisa angka negatif). input juga tidak boleh kosong

Terdapat 2 buah function, yaitu UbahWarna dan CetakPersegiPanjang.

function UbahWarna membutuhkan 1 parameter yaitu $warna. parameter $warna akan di ubah oleh switch case menjadi sebuah kode warna hexadecimal. kode warna tersebut yang akan di return oleh fungsi ini.

function CetakPersegiPanjang mebutuhkan 3 parameter yaitu $panjang, $lebar,dan $warna. didalamnya terdapat Perulangan yang digunakan untuk membuat tabel secara otomatis sesuai $panjang dan $lebar yang dimasukan.
Perulangan pertama untuk membuat baris, perulangan kedua yang berada di dalam perulangan pertama digunakan untuk membuat kolom. baris diulang sebanyak $lebar, dan kolom sebanyak input $panjang. warna dari kolom ditentukan berdasarkan parameter $warna
